# Todo List

This is a simple **Todo List web application** made using **HTML, CSS, and JavaScript**.
It helps users add and manage their daily tasks.

## Features

* Add new tasks
* Mark tasks as completed
* Edit tasks
* Delete tasks
* Tasks are saved using browser local storage

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
